# NLP Text Preprocessing Basics 🧹

This repository contains fundamental text preprocessing techniques for Natural Language Processing (NLP) pipelines using Python. 

## 🚀 Features
- Removing extra whitespaces
- Lowercasing text
- Removing punctuation marks
- Cleaning special characters using Regular Expressions (Regex)
- Basic spelling correction via TextBlob
- Extracting raw text from HTML tags using BeautifulSoup

## 🛠️ Libraries Used
- `re`
- `string`
- `textblob`
- `bs4` (BeautifulSoup)

## 💻 How to Run
Simply run the Python file to see the step-by-step console outputs of the cleaned text examples.
